Oxylus ERP
==========

This package provide `Oxylus <https://github.com/oxylus-tech/oxylus>`__ applications for enterprises and organisation management.


It features:

- `ox_erp.locations`: countries, currencies, iban and phones;
- `ox_erp.contacts`: contacts, organisations, legal entity types (per country), contact lists;

To come:

- `ox_erp.contacts_mails`: send to mails to contacts, contact list, subscriptions lists;
- `ox.apps.fin`: accounting;
- `ox.apps.cms`: CMS
