# Oxylus assets

This is the documentation for the client-side application part of Oxylus.


## Overview

The packages of Oxylus are scoped under `@oxylus`. The core library resides in `@oxylus/ox`. All other packages are related to an actual Django application.

We use the following tools: `vue`, `vuetify`, `pinia-orm` and `@pinia-orm/axios`.

Packages follows this structure:

- `components`: Vue components;
- `composables`: Vue composables;
- `models`: pinia-orm models;
- `index`: entrypoint of the application (or library);
